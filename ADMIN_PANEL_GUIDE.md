## 📑 Три Вкладки в Админ-панели

### Вкладка 1️⃣: Users (Пользователи)

**Что видно:**
- Список всех пользователей
- ID каждого пользователя
- Email
- Текущая роль (admin или user)
- Дата создания

**Что можно делать:**
- 👁️ **View** - Посмотреть детали пользователя
- 🗑️ **Delete** - Удалить пользователя (удалит и его ссылки!)
- 👑 **Change Role** - Сделать админом или убрать админ статус

**Пример:**
```
Нажать Delete на пользователе "alice":
→ alice удалена
→ Все её ссылки удалены
→ Все её клики удалены (cascade)
```

---

### Вкладка 2️⃣: Links (Ссылки)

**Что видно:**
- Все ссылки в системе (от всех пользователей)
- Короткий код ссылки
- Оригинальный URL
- Количество кликов
- Статус (active/blocked/inactive)
- ID пользователя, создавшего ссылку

**Что можно делать:**
- 🔄 **Change Status** - Выбрать статус (active/blocked/inactive)
- 🗑️ **Delete** - Удалить ссылку

**Пример:**
```
Ссылка xxUkXUH создана юзером #4 (demo)
Можно:
  - Заблокировать её: Status → blocked
  - Удалить: Delete
```

---

### Вкладка 3️⃣: User Details (Детали Пользователя)

**Что видно:**
- Информация о конкретном пользователе
- ID, Email, Username, Role
- Дата регистрации
- ВСЕ ссылки, созданные этим пользователем

**Как использовать:**
1. Перейти на вкладку
2. Выбрать пользователя
3. Видеть его профиль
4. Видеть все его ссылки
5. Удалить пользователя или изменить роль

**Пример:**
```
Выбран пользователь "demo":

- Role: user
- Создана: Mar 16, 1:26 PM
```

## 🎬 Быстрый Сценарий Тестирования

### Сценарий 1: Создать ссылку обычным пользователем

```
1. Откройте http://localhost:5175
2. Нажмите Register
3. Введите:
   Username: test1
   Password: 123
4. Нажмите Register
5. На главной введите URL: https://github.com
6. Нажмите "Generate"
7. Получите короткую ссылку! ✅
8. Откройте Dashboard - видите свою ссылку
9. Logout
```

### Сценарий 2: Управлять ссылками админом

```
1. Login как admin (admin@short-link.com / Admin@1234)
2. Откройте http://localhost:5175/admin
3. Перейдите на вкладку "Links"
4. Найдите ссылку xxUkXUH (создана test1)
5. Нажмите "Change Status"
6. Выберите "blocked"
7. Ссылка теперь заблокирована! ✅
8. Пользователь не сможет её открыть
```

### Сценарий 3: Удалить пользователя

```
1. Login как admin
2. Откройте http://localhost:5175/admin
3. Перейдите на вкладку "Users"
4. Найдите пользователя "alice"
5. Нажмите "Delete"
6. ✅ alice удалена
7. ✅ Все её ссылки удалены
8. ✅ Все её клики удалены
```

### Сценарий 4: Сделать админом

```
1. Login как admin
2. Откройте http://localhost:5175/admin
3. Перейдите на вкладку "Users"
4. Найдите пользователя "bob"
5. Нажмите "Change Role"
6. Выберите "admin"
7. ✅ bob теперь админ!
8. Logout и Login как bob
9. Откройте /admin - bob может управлять!
```

## 💡 Важные Замечания

### Безопасность
- ✅ Админ-панель защищена (требует логина админа)
- ✅ Токены хранятся в localStorage (7 дней)
- ✅ Пароли хешированы bcryptjs
- ✅ Cascade delete (при удалении юзера удаляются его ссылки и клики)

### Пароли теперь проще
- ✅ Email **опционален** (автогенерируется)
- ✅ Пароль минимум **3 символа** (было 6)
- ✅ **Не нужна сложность** (спецсимволы, цифры, буквы)
- ✅ **Без подтверждения пароля**

### Примеры простых паролей
```
✅ Валидные пароли:
  - "123"
  - "abc"
  - "pass"
  - "mypass123"
  - "qwerty"

❌ Невалидные пароли:
  - "12" (меньше 3 символов)
  - "" (пусто)
```

## 🆘 Решение Проблем

### Админ-панель не загружается
- **Проверка 1:** Вы залогинены как админ? (username admin)
- **Проверка 2:** Откройте F12 → Console → есть ошибки?
- **Решение:** Logout → Login снова

### Не могу заблокировать ссылку
- **Причина:** Вы не админ
- **Решение:** Login как admin@short-link.com

### Пользователь не удаляется
- **Проверка:** Есть в БД пользователь?
- **Решение:** Refresh страницу (F5)

### Ссылка не удаляется
- **Проверка:** Вы админ?
- **Решение:** Перезагрузите админ-панель
