# Улучшения Версии 1.1.0 - 15 Марта 2026

## ✅ Реализованные улучшения

### 1. **Отслеживание источников кликов (Statistics Page)**
- ✅ Добавлены отдельные счётчики для кликов через QR-код и прямую ссылку
- ✅ Backend отслеживает параметр `source` (значения: `qr`, `direct`)
- ✅ База данных обновлена: добавлена колонка `source` в таблицу `clicks`
- ✅ API возвращает `sourceStats` с детальной статистикой по источникам

**Нововведение на синтаксис ссылок:**
```
http://localhost:3000/{shortCode}?source=qr      # Клик из QR-кода
http://localhost:3000/{shortCode}?source=direct  # Прямая ссылка
http://localhost:3000/{shortCode}                # По умолчанию - direct
```

**На странице Statistics отображаются:**
- 📊 Total Clicks - Всего кликов
- 📱 Clicks via QR Code - Клики по QR коду
- 🔗 Clicks via Direct Link - Клики по прямой ссылке

## 📊 Примеры использования

### Создание ссылки с отслеживанием QR
```javascript
// На frontend при генерации QR-кода:
const qrUrl = `http://localhost:3000/${shortCode}?source=qr`;

// Пользователь сканирует QR → система записывает source=qr
```

### Получение статистики
```bash
curl http://localhost:3000/api/links/7/stats
```

**Ответ:**
```json
{
  "clickCount": 4,
  "sourceStats": [
    {"source": "qr", "count": "2"},
    {"source": "direct", "count": "2"}
  ]
}
```
